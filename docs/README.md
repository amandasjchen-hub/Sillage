# Sillage — Company Documentation

This folder contains the decision history, architecture notes, and context behind every major choice made while building Sillage. It is written so that any future employee, developer, or collaborator can understand not just *what* was built, but *why*.

## How to use this folder

Every time a significant decision is made — a new feature, a technology choice, a change in direction — a new document is added here with the date, the decision, and the reasoning. Nothing is deleted; the history is permanent.
